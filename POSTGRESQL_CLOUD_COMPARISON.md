## 🚀 快速部署指南

### Railway 部署（推荐）

```bash
# 1. 安装 CLI
npm install -g @railway/cli

# 2. 登录
railway login

# 3. 初始化
cd mingzhita-main
railway init

# 4. 添加 PostgreSQL
railway add postgresql

# 5. 设置环境变量（自动）
# DATABASE_URL 会自动注入

# 6. 部署
railway up

# 完成！访问自动生成的域名
```

### Render 部署

```
1. 访问 render.com
2. 点击 "New +" → "Web Service"
3. 连接 GitHub 仓库
4. 配置：
   - Build Command: npm install && npm run build
   - Start Command: npm run start
5. 添加 PostgreSQL：
   - 点击 "New +" → "PostgreSQL"
   - 连接到 Web Service
6. 自动部署完成
```

### Supabase + Vercel 部署

```
1. 注册 Supabase.com
2. 创建项目，获取 DATABASE_URL
3. 注册 Vercel.com
4. 连接 GitHub 仓库
5. 配置环境变量：
   - DATABASE_URL=你的 Supabase 连接
6. 自动部署
```
