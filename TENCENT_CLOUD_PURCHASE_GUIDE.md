# 腾讯云服务器购买详细教程

## 📋 购买清单

你需要购买两样东西：
1. **轻量应用服务器**（运行应用）- ¥74/月
2. **云数据库 MySQL**（存储数据）- ¥30/月

**总计：¥104/月**

---

## 第一部分：购买轻量应用服务器（10 分钟）

### 步骤 1：注册腾讯云账号

1. 访问 [腾讯云官网](https://cloud.tencent.com/)
2. 点击右上角 **"注册"**
3. 选择注册方式：
   - 微信扫码（推荐，最快）
   - 手机号注册
   - QQ 登录
4. 完成注册

### 步骤 2：实名认证（必须）

1. 登录后，点击右上角头像
2. 选择 **"账号信息"**
3. 点击 **"实名认证"**
4. 选择认证类型：
   - **个人认证**（推荐）
     - 需要：身份证正反面照片
     - 审核时间：5-10 分钟
   - **企业认证**
     - 需要：营业执照
     - 审核时间：1-3 天

5. 上传身份证照片
6. 填写个人信息
7. 提交审核
8. 等待审核通过（通常 5-10 分钟）

**💡 提示：可以先继续下面的步骤，审核通过后再付款**

### 步骤 3：购买轻量应用服务器

#### 3.1 进入购买页面

1. 访问 [轻量应用服务器购买页](https://buy.cloud.tencent.com/lighthouse)
2. 或者：腾讯云首页 → 产品 → 计算 → 轻量应用服务器 → 立即选购

#### 3.2 选择配置

**地域选择：**
```
推荐选择：广州
原因：
  - 国内访问速度快
  - 稳定性好
  - 价格便宜

其他选项：
  - 上海：适合华东用户
  - 北京：适合华北用户
  - 成都：适合西南用户
```

**镜像选择：**
```
选择：系统镜像 → Ubuntu → Ubuntu 20.04 LTS

不要选：
  - 应用镜像（我们自己安装）
  - Windows（贵且不适合）
```

**套餐选择：**
```
推荐：2核 4GB 6Mbps 80GB SSD

配置详情：
  - CPU：2 核心
  - 内存：4GB
  - 带宽：6Mbps（峰值）
  - 流量：1200GB/月（够用）
  - 硬盘：80GB SSD
  - 价格：¥74/月

为什么选这个：
  - 性价比最高
  - 支持 800-1500 并发
  - 流量充足
```

**购买时长：**
```
建议：先买 1 个月测试

选项：
  - 1 个月：¥74（测试用）
  - 3 个月：¥222（无优惠）
  - 6 个月：¥444（无优惠）
  - 1 年：¥888（无优惠）

💡 提示：
  - 先买 1 个月测试
  - 确认没问题后再续费
  - 续费时可能有优惠活动
```

#### 3.3 确认订单

1. 检查配置：
   - 地域：广州
   - 镜像：Ubuntu 20.04
   - 套餐：2核4GB 6Mbps
   - 时长：1 个月
   - 价格：¥74

2. 勾选 **"我已阅读并同意"**

3. 点击 **"立即购买"**

#### 3.4 支付

1. 选择支付方式：
   - 微信支付（推荐）
   - 支付宝
   - QQ 钱包
   - 银行卡

2. 完成支付

3. 等待创建完成（约 1-2 分钟）

### 步骤 4：获取服务器信息

#### 4.1 进入控制台

1. 支付成功后，点击 **"进入控制台"**
2. 或访问 [轻量应用服务器控制台](https://console.cloud.tencent.com/lighthouse/instance)

#### 4.2 查看服务器信息

你会看到你的服务器，显示：
```
实例名称：lighthouse-xxxxx
状态：运行中
公网 IP：123.456.789.0（这个很重要！）
配置：2核4GB
```

#### 4.3 设置密码

1. 点击服务器名称进入详情
2. 找到 **"重置密码"** 按钮
3. 设置一个强密码（记住它！）
   - 至少 8 位
   - 包含大小写字母、数字、特殊字符
   - 例如：`Abc123!@#`
4. 点击 **"确定"**
5. 等待重置完成（约 1 分钟）

#### 4.4 记录信息

**重要！记录以下信息：**
```
公网 IP：123.456.789.0
用户名：ubuntu（Ubuntu 系统默认）
密码：你刚设置的密码
```

## 第二部分：购买云数据库 MySQL（5 分钟）

### 步骤 1：进入购买页面

1. 访问 [云数据库 MySQL 购买页](https://buy.cloud.tencent.com/cdb)
2. 或者：腾讯云首页 → 产品 → 数据库 → 云数据库 MySQL → 立即选购

### 步骤 2：选择配置

**计费模式：**
```
选择：包年包月（推荐）

不要选：
  - 按量计费（贵，适合临时测试）
```

**地域和可用区：**
```
地域：广州（与服务器相同）
可用区：随意（默认即可）

⚠️ 重要：必须与服务器在同一地域！
```

**数据库版本：**
```
选择：MySQL 5.7 或 MySQL 8.0

推荐：MySQL 8.0
  - 性能更好
  - 功能更多
  - 长期支持
```

**架构：**
```
选择：基础版（推荐）

说明：
  - 基础版：单节点，便宜
  - 高可用版：双节点，贵一倍

对于个人项目，基础版够用
```

**规格：**
```
选择：1核 2GB 内存

配置详情：
  - CPU：1 核心
  - 内存：2GB
  - 存储：20GB
  - 价格：¥30/月

为什么选这个：
  - 最便宜
  - 性能够用（支持 800-1500 并发）
  - 可以随时升级
```

**硬盘：**
```
选择：20GB（默认）

说明：
  - 20GB 够用（数据库不会太大）
  - 后续可以扩容
```

**网络：**
```
选择：默认 VPC（默认即可）

说明：
  - 会自动与服务器在同一网络
  - 可以通过内网连接（免费、快速）
```

**购买时长：**
```
建议：1 个月

原因：
  - 先测试
  - 确认没问题再续费
```

### 步骤 3：确认订单

1. 检查配置：
   - 地域：广州
   - 版本：MySQL 8.0
   - 规格：1核2GB 20GB
   - 时长：1 个月
   - 价格：¥30

2. 勾选 **"我已阅读并同意"**

3. 点击 **"立即购买"**

### 步骤 4：支付

1. 选择支付方式
2. 完成支付
3. 等待创建完成（约 3-5 分钟）

### 步骤 5：配置数据库

#### 5.1 进入控制台

1. 访问 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)
2. 找到你的数据库实例

#### 5.2 初始化数据库

1. 点击实例名称进入详情
2. 如果显示 **"未初始化"**，点击 **"初始化"**
3. 设置：
   - **字符集**：utf8mb4（支持中文和 emoji）
   - **表名大小写**：不敏感（推荐）
   - **root 密码**：设置一个强密码（记住它！）
4. 点击 **"确定"**
5. 等待初始化完成（约 1-2 分钟）

#### 5.3 创建数据库

1. 点击 **"数据库管理"** 标签
2. 点击 **"创建数据库"**
3. 填写：
   - **数据库名**：`mingzhita`
   - **字符集**：utf8mb4
   - **排序规则**：utf8mb4_general_ci
4. 点击 **"确定"**

#### 5.4 创建账号（可选但推荐）

1. 点击 **"账号管理"** 标签
2. 点击 **"创建账号"**
3. 填写：
   - **账号名**：`mingzhita`
   - **主机**：`%`（允许所有 IP）
   - **密码**：设置一个强密码（记住它！）
   - **备注**：应用账号
4. 点击 **"确定"**
5. 点击 **"修改权限"**
6. 选择数据库：`mingzhita`
7. 勾选 **"读写权限"**
8. 点击 **"确定"**

#### 5.5 记录数据库信息

**重要！记录以下信息：**

1. 点击实例名称进入详情
2. 找到 **"内网地址"**（例如：`10.0.0.123:3306`）
3. 记录：

```
内网地址：10.0.0.123
端口：3306
数据库名：mingzhita
用户名：mingzhita（或 root）
密码：你设置的密码
```

**⚠️ 重要：使用内网地址，不要用外网地址！**
- 内网：免费、快速、安全
- 外网：收费、慢、不安全

## 第三部分：配置安全组（5 分钟）

### 步骤 1：配置服务器防火墙

1. 进入 [轻量应用服务器控制台](https://console.cloud.tencent.com/lighthouse/instance)
2. 点击你的服务器
3. 点击 **"防火墙"** 标签
4. 点击 **"添加规则"**

**添加以下规则：**

**规则 1：SSH（已有，检查即可）**
```
应用类型：自定义
协议：TCP
端口：22
策略：允许
来源：0.0.0.0/0
备注：SSH 登录
```

**规则 2：HTTP**
```
应用类型：自定义
协议：TCP
端口：80
策略：允许
来源：0.0.0.0/0
备注：HTTP
```

**规则 3：HTTPS**
```
应用类型：自定义
协议：TCP
端口：443
策略：允许
来源：0.0.0.0/0
备注：HTTPS
```

**规则 4：宝塔面板**
```
应用类型：自定义

端口：8888
策略：允许
来源：0.0.0.0/0
备注：宝塔面板
```

**规则 5：应用端口（可选）**
```
应用类型：自定义
协议：TCP
端口：3000
策略：允许
来源：0.0.0.0/0
备注：应用端口
```

5. 点击 **"确定"**

## 📞 遇到问题？

### 常见问题：

**Q1：实名认证一直不通过？**
- 检查身份证照片是否清晰
- 确认信息填写正确
- 联系腾讯云客服

**Q2：支付失败？**
- 检查账户余额
- 更换支付方式
- 联系银行客服

**Q3：服务器创建失败？**
- 刷新页面重试
- 更换地域重试
- 联系腾讯云客服

**Q4：数据库连接不上？**
- 确认使用内网地址
- 检查账号密码
- 检查数据库是否初始化

### 联系客服：

- 腾讯云客服电话：95716
- 在线客服：腾讯云控制台右下角
- 工单系统：控制台 → 工单 → 提交工单
