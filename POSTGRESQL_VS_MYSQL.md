## 🔧 代码改动对比

### 如果你已经写了 PostgreSQL 代码

#### 改为 MySQL 需要修改：

**1. 数据类型差异**
```typescript
// PostgreSQL
model User {
  id        Int      @id @default(autoincrement())
  data      Json     // PostgreSQL JSON 类型
  createdAt DateTime @default(now())
}

// MySQL
model User {
  id        Int      @id @default(autoincrement())
  data      Json     @db.Json // MySQL 需要指定
  createdAt DateTime @default(now())
}
```

**2. 环境变量**
```env
# PostgreSQL
DATABASE_URL=postgresql://user:pass@host:5432/db

# MySQL
DATABASE_URL=mysql://user:pass@host:3306/db
```

**3. Prisma Schema**
```prisma
// PostgreSQL
datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}

// MySQL
datasource db {
  provider = "mysql"
  url      = env("DATABASE_URL")
}
```

**改动量：很小（5 分钟）**
